NOT A TREK FAN?
The LCARS aesthetic is the default — it's what On Screen looks like out of the box. But it's one of four bundled themes and takes about 10 seconds to change. Open settings with , → THEMES tab → pick VANILLA, PAPER, or TERMINAL. Or build your own with the color picker. Everything is configurable. The app is the function — the look is just a starting point.

CORE SYSTEMS
DESKTOP LAYER

Runs behind all windows at the X11 desktop layer. The OS sees it as wallpaper. You see a pinboard. Toggle edit mode with E and it becomes interactive.

CARDS
  • Text notes — inline editing, titled, color-coded
  • Image cards — URL, clipboard paste, or file drop
  • Drag from header strip, resize from corner
  • Infinite canvas, pan with Space + drag
THEMES
  • 4 bundled themes out of the box
  • Simple 5-color picker — borders and grid auto-derived
  • Full CSS var editor for power users
  • Save, name, and recall custom themes
  • Random palette generator — one click
EXPORT / IMPORT

Your board is plain JSON — readable, hackable, version-controllable. Export to back up or share layouts. Import to restore. Works across machines.

EFFECTS
  • All effects off by default — zero idle cost
  • CSS transitions, drag lift, hover glow
  • Card appear animation, ambient pulse
  • Enable exactly what you want
STACK
  • Tauri shell — Rust + WebKit
  • Zero JS frameworks, zero CDN calls
  • Single HTML file core — auditable
  • JSON persistence, no database
